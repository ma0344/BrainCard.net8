# Contributing

## UI/UX 方針

### ロード中の入力抑止と表示

ロードなど時間のかかる処理中は、必ず以下の挙動にします。

- 入力抑止: 画面全体のオーバーレイを表示し、ユーザー操作を受け付けない
- 理由表示: オーバーレイ中央付近に「ロード中…」等のメッセージとインジケータを表示する
- 例外安全: ロード処理は `try/finally` で終了時にオーバーレイを必ず解除する
- 多重実行防止: ロード処理は二重に開始されないよう排他制御する

## コーディング規約

- コメントは日本語で記述してください。
- UI表示文言は `Properties/Resources.*.xaml` のリソースキーとして管理してください。
